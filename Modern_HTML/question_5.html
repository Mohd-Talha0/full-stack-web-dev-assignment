HTML5 Feature Presentation

This page gives a brief illustration of few features of the HTML5.

Feature 1: Form Validation

HTML5 offers a built-in form validation without the need for a custom JavaScript code.



Feature 2: Video and audio playback

The HTML5 allows the direct embedding of the videos and audios in web pages without the need for plugins such as flash.

Feature 3: Semantic Tags

HTML5 has introduced semantic tags such as <header>,<nav>, <footer>, and more, which improves the page structure and accessibility.
